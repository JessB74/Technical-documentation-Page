HTML & CSS Documentation

Introduction

HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are essential tools for creating web pages. HTML defines the structure and content of a webpage, while CSS is responsible for its visual appearance and layout.

HTML Basics

HTML is the standard markup language used to create web pages. It describes the structure of a web page using a variety of elements and attributes.

  • Elements: The building blocks of HTML.
  • Tags: Define the start and end of an HTML element.
  • Attributes: Provide additional information about elements.

CSS Basics

CSS is a style sheet language used for describing the look and formatting of a document written in HTML. CSS allows you to apply styles to web pages.

  • Selecting elements: Target HTML elements to style using CSS selectors.
  • Styling properties: Apply properties like color, font, and layout to selected elements.
  • Responsive design: Make web pages look good on all devices.
HTML Elements

HTML elements are the building blocks of HTML pages. Examples include headings, paragraphs, links, images, and lists.

<html>
<head>
<meta charset="utf-8">
<linkrel="stylesheet" href="styles.css>
<title>This is a title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<a href="https://www.example.com">Example</a>
</body>
</html>
CSS Selectors

CSS selectors are patterns used to select the elements you want to style.

  • Element Selector: Selects elements by name. Example: p{color:red}
  • Class Selector: Selects elements by class name. Example: .classname{background-color:blue}
  • ID Selector: Selects elements by ID. Example: #idname{font-size:20px}
  • Attribute Selector: Selects elements based on an attribute. Example: input[type=text]{width:20px}
  • Universal Selector:Selects all elements. Example: *{box-sizing:border-box}
CSS Box Model

The CSS box model is a fundamental concept that determines the size, padding, border, and margin of HTML elements.

  • Content: The actual content of the box, where text and images appear.
  • Padding: Clears an area around the content.
  • Border: A border that goes around the padding and content.
  • Margin: Clears an area outside the border.
CSS Layout

CSS provides several layout mechanisms for creating the overall layout of your web pages, including Flexbox, Grid, and positioning techniques.

/* Example code of a Flexbox layout */
.container {
    display: flex;
    justify-content: space-around;
    align-items: center;
}
/* Example code of a Grid layout */
.container {
    display: grid;
    grid-template-columns:1fr 2fr auto;
    grid-template-rows:2px minmax(20px, 40px);
    justify-content: space-around;
    align-items: center;
}
/* Example code of  CSS position */
.container {
   position: relative;
    top:30px;
    bottom: 0;
   left:150px;
   right: 40px;
}
Responsive Design

Responsive design is a crucial aspect of modern web development, ensuring that websites look and function well on a variety of devices, from desktop computers to tablets and smartphones. The goal is to create a seamless user experience regardless of screen size or resolution.
Bootstrap simplifies this process with its grid system and responsive components.
Flexbox offers flexible, one-dimensional layouts that easily adjust to screen changes.
CSS Grid provides more control for creating complex, two-dimensional layouts.
Together, these tools help developers create visually consistent and functional web pages on any device.

/* Example of a media query */
@media (max-width: 600px) {
    .container {
        flex-direction: column;
    }
}

 </div>
</div>
